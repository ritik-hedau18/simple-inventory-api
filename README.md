# Warehouse Inventory API

Warehouse Inventory API is a Spring Boot REST API service managing Inventory records.

## Features
- Stock items tracking
- Warehouse metadata tags
- Low stock alerting logs

## Running Locally
Run `mvn spring-boot:run` in the project directory. The service runs on port `8080`.
